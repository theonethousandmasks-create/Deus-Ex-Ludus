# Deus-Ex-Ludus

Game under development from FoxMaid Games. Become the deity you've always dreamed of in this TTRPG where you guide your people to glory.

## Foundry VTT System

This repository contains the Foundry VTT system implementation for Deus Ex Ludus.

### Features

- d100 target number system
- Degrees of success/failure based on increments of 10
- Character sheets with skills
- Item sheets for skills, weapons, armor

### Installation

1. Download or clone this repository.
2. Place the `Deus-Ex-Ludus` folder in your Foundry VTT `Data/systems` directory make sure it has the name `Deus-Ex-Ludus` with no other additions.
3. Launch Foundry VTT and select "Deus Ex Ludus" as the system for a new world.

### Usage

- Create actors of type "character" or "npc".
- Set skill target numbers (TN) from 0-100.
- Click the "Roll" button next to a skill to roll d100 and determine success/failure with degree.
